# CS2 Marketplace IDs

Most tradable CS2 in-game items' Name/ID pairs from various item marketplaces such as [buff163](https://buff.163.com), [Youpin898 (UU)](https://www.youpin898.com/), [buff.market](https://buff.market/), and [cs.money](https://cs.money/) can be found here including [Jackass Sticker Capsule](https://store.steampowered.com/news/app/730/view/701021228894257190), [IEM Cologne 2026](https://store.steampowered.com/news/app/730/view/672869045073084948) and [NIGHTMODE II Music Kits](https://store.steampowered.com/news/app/730/view/702141174212725164) items. All items' Goods IDs, Applied Sticker IDs (including exclusively pre-applied stickers), Applied Patch IDs, Applied Charm/Sticker Slab IDs, Doppler Phase IDs and some items' pattern tiers can be found within a single JSON file.<br>

Some known missing items (including stickers from the IEM Cologne 2026 Major), Doppler phases and Fire and Ice pattern tier tag IDs for certain Marble Fade knives which aren't registered on buff163, Youpin898, buff.market, or cs.money are listed on [knownmissing.txt](https://github.com/ModestSerhat/cs2-marketplace-ids/blob/main/knownmissing.txt) and [knownmissingslabs.txt](https://github.com/ModestSerhat/cs2-marketplace-ids/blob/main/knownmissingslabs.txt) for each marketplaces' missing sticker slab IDs.

Some stickers and patches can have other Sticker/Patch ID-Name pairs on buff163 with the name being the same that have a lower quantity of listings of the same kind of sticker applied (e.g. Rush 4x20 (Holo), CRUC1AL | Paris 2023)<br>
